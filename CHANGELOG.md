# Changelog

All notable changes to this skill are recorded here. Newest first.

## [2.0.0] — 2026-07-15

First public release.

- Searches the skillsmp.com catalog for a topic and reads each result's real `SKILL.md` to detect Cowork / Claude Desktop support.
- Dedupes copies and forks, keeping the most-starred version of each skill.
- Ranks by GitHub stars, cowork-ready first, and writes a single branded HTML report.
- Marks skills published by known vendors and separates cowork-ready results from the rest.
- No account, login, or token required — public data over plain HTTP.
